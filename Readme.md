# Sales Data Analysis Dashboard (Power BI)

## Project Overview
This project is an interactive Power BI dashboard built using a sales dataset to analyze category wise performance and monthly sales trends.

## Tools Used
Power BI
Excel

## Key Insights
December recorded the highest sales indicating strong seasonal demand.
Category-wise analysis shows Technology as a top performing segment.
Interactive slicer enables dynamic data filtering.
